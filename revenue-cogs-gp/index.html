Revenue · COGS · Gross Profit Report

Upload the Sales (Revenue) transaction report and the COGS transaction report → get a clean 3-tab Profit & Loss report with Gross Profit, Net Profit and margins.

🔒 Files are processed entirely in your browser. Nothing is uploaded to any server.

1 Upload your two transaction reports

Both files are standard QuickBooks Transaction Reports (the ones with a Transaction type / Name / Amount layout). The app auto-detects the header row, data rows and the report total.

2 Add operating expenses (for Net Profit)

Your two files contain Revenue and COGS only. To show Net Profit and the Net Profit margin, enter the total operating expenses below the gross-profit line (payroll, rent, marketing, admin, etc.). Leave it at 0 if you only want Gross Profit.

Stored only in this browser. How do I get this?
One-time setup (Google Cloud, ~3 min):
  1. Go to console.cloud.google.com → APIs & Services → Enable the Gmail API.
  2. OAuth consent screen → Internal (your Workspace) → add scope gmail.compose.
  3. Credentials → Create OAuth client ID → Web application.
  4. Under Authorized JavaScript origins add this page's URL (e.g. https://revenue-cogs-gp-report.vercel.app).
  5. Copy the Client ID and paste it above.
