Remote Deposit Check Extractor

Drop in scanned check PDFs → get a remote-deposit import sheet: Date, Description (check #), Amount.

🔒 Files are read and OCR'd entirely in your browser. Nothing is uploaded to any server.

1 Upload the check PDFs

Add one or more scanned check PDFs. Each PDF may contain several checks — every check becomes its own row.

2 Extract the check details

The tool renders every page, runs OCR, and pulls the check date, check number, and amount. Backs of checks (no amount) are skipped automatically.

Add at least one PDF to enable.

3 Review & download

Each row shows the check image so you can confirm the values. Fix anything flagged, then download. Rows in amber need a look.
